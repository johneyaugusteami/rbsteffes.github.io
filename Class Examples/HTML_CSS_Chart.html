These are the HTML Elements to Know

HTML

Useful HTML tags
HTML Tag When to use Is Semantic Special Attributes
Inline Objects
<b> when things should be bold no no
<em> when things should be emphasized no no
<span> when things need to be selected no no
<a> when things should be linked no
  • href = the address the thing goes to
  • target = top is same page, _blank it's new page
<img> to show images no
  • src = the location of the image
  • alt = the USEFUL description of the image
Block Objects
<div> when you need a box no no
<main> usually the main portion of the webpage no no
<article> the main content of a webpage no no
<header> content intended to serve as a banner for the page no no
<footer> information intended for the bottom, usually social media and links no no
<aside> information only loosely related to the main content no no

CSS

Useful CSS properties
Property Effect
font-size changes font size
font-weight make font bold
width changes width of block elements
height changes height of block elements
position
  • static -- unchanged, can't move, "in the flow of the document"
  • relative -- displayed relative to starting position
  • absolute -- displayed relative to page or positioned ancestor
  • fixed -- relative to viewscreen window
float Removes an element from flow, wraps other elements around it
clear stops from being positioned by a float
text-align Aligns content text
display none -- hidden, block, or inline
margin minimum distance between elements
border wrapped around padding, takes width, style and color
padding distance between content and border
